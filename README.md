# 禄口畅行

南京禄口机场移动端导航与机场周边导购原型。

## 导航说明

- “导航至禄口机场”和“导航至机场地铁站”会调用高德地图 URI API，从手机当前位置规划真实路线。
- 登机口、地铁站与机场商户提供虚拟室内导航演示：使用虚拟路网计算最短路径，并模拟 BLE/Wi-Fi 定位、楼层切换、距离变化与逐步导航。
- 演示地图依据南京禄口国际机场官网公开设施分布图进行简化重绘，并非精确测绘地图。
- 官方设施图：https://www.njiairport.com/web/about/21.html

## 本地打开

直接打开 `index.html`，或使用任意静态文件服务器运行。

## 手机访问

网站可通过 GitHub Pages 从 `main` 分支根目录直接部署。
